Experiment 1: Basic HTML Page

Objective: Create a simple HTML page with headings, paragraphs and lists.

Steps:

  1. Create a basic HTML5 document structure
  2. Add headings (h1-h6)
  3. Create ordered and unordered lists

Experiment 2: CSS Styling

Objective: Apply CSS styles to HTML elements.

Steps:

  1. Create external CSS file
  2. Style text (font, color, size)
  3. Add borders and backgrounds

Experiment 3: JavaScript Form Validation

Objective: Validate form inputs using JavaScript.

Steps:

  1. Create a registration form
  2. Add validation for email format
  3. Validate password strength

Experiment 4: HTML Tables and Forms

Objective: Design a form and display the data using HTML tables.

Steps:

  1. Create a user input form
  2. Collect data using input fields
  3. Use a table to format and display the data

Experiment 5: Responsive Web Page

Objective: Design a responsive webpage using CSS media queries.

Steps:

  1. Create a multi-section layout
  2. Use media queries for responsiveness
  3. Test on different screen sizes

Experiment 6: Navigation Bar with CSS

Objective: Create a horizontal navigation bar using only CSS.

Steps:

  1. Create nav elements using <ul> and <li>
  2. Style them horizontally with CSS
  3. Add hover effects

Experiment 7: DOM Manipulation

Objective: Modify HTML content using JavaScript DOM methods.

Steps:

  1. Access elements using getElementById
  2. Change text and styles dynamically
  3. Handle button click events

Experiment 8: Image Gallery with CSS Grid

Objective: Create a responsive image gallery using CSS Grid.

Steps:

  1. Insert multiple images into a grid layout
  2. Apply CSS Grid properties
  3. Make it responsive using media queries

Experiment 9: Mini Project - Personal Portfolio

Objective: Build a personal portfolio website with HTML, CSS, and JavaScript.

Steps:

  1. Design sections: About, Projects, Contact
  2. Add interactive elements using JavaScript
  3. Host the portfolio on GitHub Pages
